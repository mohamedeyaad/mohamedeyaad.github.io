Stepper Motor GUI

This is a Project was part of the Electromechanical Systems Course @ UCLan (University of Central Lanchasire) in Preston, UK. It was during my exchange program which is funded and supported by Erasmums+. The aim of the project is to control the speed and the direction of 2 steppper motors using a GUI (Graphical User Interface). I have used LabVIEW software to design the GUI and interfaced it with Arduino to give signals for the motor driver to control the motors.
