Rehablitation Robotics Project

This is a Project was part of the "Rehablitation Robotics" Course @ Faculty of Engineering - Ain Shams University, Egypt.
The main target of the project is to design and implement a control testing rig to be used in differentiating between different control techniques. The device was made with a single degree of freedom using a DC motor and sensors such as encoder, load ccell, current sensor.

The following control techniques were applied:
  • Position Control
  • Force Control
  • Current Control
  • Admittance Control
  • Impedence Control
